# 挑战1：设计一个计算器程序

## 难度
⭐⭐

## 描述

你已经学会了冯诺依曼架构的基本原理。现在，请设计一个简单的计算器程序，能够执行两个数的加、减、乘、除运算。

## 要求

1. **设计指令集**：
   - 需要支持的基本指令：LOAD, ADD, SUB, MUL, DIV, STORE, HALT
   - 为每条指令分配操作码（4位）

2. **内存布局**：
   - 地址0-9：程序代码区
   - 地址10-12：操作数A、B和运算符（1=加, 2=减, 3=乘, 4=除）
   - 地址20：结果存储区

3. **实现功能**：
   - 根据地址12的运算符，对A和B进行相应运算
   - 将结果存入地址20

4. **示例**：
   - 如果 内存[10]=5, 内存[11]=3, 内存[12]=1
   - 则执行 5 + 3 = 8
   - 结果 内存[20] = 8

## 输入

内存预置：
- 内存[10] = 第一个操作数（A）
- 内存[11] = 第二个操作数（B）
- 内存[12] = 运算符代码（1=加, 2=减, 3=乘, 4=除）

## 输出

- 内存[20] = 计算结果

## 示例

```
输入:
  内存[10] = 8
  内存[11] = 4
  内存[12] = 2  (减法)

输出:
  内存[20] = 4  (8 - 4 = 4)
```

## 提示

- 思考如何用条件跳转实现多分支（类似switch/case）
- 除法要注意除数为0的情况
- 可以先从简单的加法开始，逐步添加其他运算

## 进阶挑战

如果你完成了基础版本，尝试添加：
1. 错误处理（如除数为0时返回错误码）
2. 支持连续运算（如 A + B - C）
3. 添加取余运算（MOD）
