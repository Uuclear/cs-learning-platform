# 挑战2：实现一个简单的排序算法

## 难度
⭐⭐⭐

## 描述

冯诺依曼架构的计算机可以执行任何算法，包括排序！请使用你学到的指令集，实现一个简单的冒泡排序算法，对内存中的数组进行排序。

## 背景知识：冒泡排序

冒泡排序的基本思想：
1. 比较相邻的两个元素
2. 如果顺序不对（前一个大于后一个），交换它们
3. 重复这个过程，直到没有交换发生（数组已排序）

就像气泡一样，大的元素会慢慢"浮"到数组的末尾。

## 要求

1. **数据准备**：
   - 在内存地址20-24存放5个待排序的数字
   - 例如：[64, 34, 25, 12, 22]

2. **实现冒泡排序**：
   - 使用两重循环（外层控制轮数，内层控制比较）
   - 使用条件跳转实现循环和条件判断
   - 实现两个数的比较和交换

3. **指令集**（在基础指令集上扩展）：
   - LOAD, STORE, ADD, SUB
   - JUMP（无条件跳转）
   - JUMP_IF_ZERO（为零跳转）
   - JUMP_IF_NEGATIVE（为负跳转）
   - HALT

4. **输出**：
   - 排序后的数组存储在原位置（地址20-24）

## 输入

```
内存[20] = 64
内存[21] = 34
内存[22] = 25
内存[23] = 12
内存[24] = 22
```

## 输出

```
内存[20] = 12
内存[21] = 22
内存[22] = 25
内存[23] = 34
内存[24] = 64
```

## 算法伪代码

```python
# 冒泡排序伪代码
n = 5  # 数组长度
for i in range(n):
    swapped = False
    for j in range(n - 1 - i):
        if arr[j] > arr[j+1]:
            # 交换 arr[j] 和 arr[j+1]
            temp = arr[j]
            arr[j] = arr[j+1]
            arr[j+1] = temp
            swapped = True
    if not swapped:
        break  # 已经有序，提前结束
```

## 提示

1. **变量分配**：
   - 内存[10]：外层循环计数器 i
   - 内存[11]：内层循环计数器 j
   - 内存[12]：是否交换的标志 swapped
   - 内存[13-14]：临时变量

2. **关键问题**：
   - 如何访问 arr[j] 和 arr[j+1]？
   - 如何实现两个数的比较？（提示：用减法，看结果是否为负）
   - 如何实现交换？（需要临时变量）

3. **内存寻址技巧**：
   - 如果 j 存储在内存[11]，arr 从地址20开始
   - 那么 arr[j] 的地址 = 20 + j
   - 但你的简单CPU可能不支持间接寻址...
