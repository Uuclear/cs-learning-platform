# 挑战 2：温度转换器

## 目标
创建一个简单的温度转换程序，能够在摄氏度和华氏度之间进行转换。

## 要求
1. 程序应该显示一个菜单，让用户选择转换方向：
   - 1: 摄氏度转华氏度
   - 2: 华氏度转摄氏度
2. 根据用户选择读取温度值
3. 执行相应的转换计算并显示结果
4. 使用函数来组织代码（即使我们还没正式学习函数，可以尝试模仿main函数的结构）

## 转换公式
- 摄氏度转华氏度: F = C × 9/5 + 32
- 华氏度转摄氏度: C = (F - 32) × 5/9

## 提示
- 使用 `loop` 创建一个可以重复运行的程序
- 使用 `match` 或 `if-else` 处理用户的选择
- 使用浮点数类型（如 `f64`）进行计算
- 考虑如何优雅地退出程序（例如输入 'q'）

## 扩展挑战（可选）
- 添加开尔文温度的支持
- 格式化输出，保留两位小数
- 添加输入验证，确保用户输入的是有效数字

## 验证标准
- 程序能够正确编译和运行
- 能够正确执行温度转换
- 用户界面友好，错误处理得当
- 代码结构清晰，易于理解
