# 挑战 1：个性化问候程序

## 目标
创建一个Rust程序，能够根据用户输入的名字显示个性化的问候信息。

## 要求
1. 使用 `std::io` 模块读取用户输入
2. 程序应该提示用户输入他们的名字
3. 显示类似 "你好，[名字]！欢迎学习Rust编程！" 的消息
4. 如果用户输入为空，显示默认消息 "你好，神秘的Rust学习者！"

## 提示
- 使用 `println!` 显示提示信息
- 使用 `std::io::stdin()` 读取输入
- 使用 `trim()` 方法去除输入两端的空白字符
- 使用 `if` 表达式检查输入是否为空

## 扩展挑战（可选）
- 让程序询问用户的年龄，并根据年龄显示不同的消息
- 添加错误处理，确保程序在输入错误时不会崩溃

## 验证标准
- 程序能够正确编译和运行
- 能够处理空输入和正常输入
- 代码格式良好，符合Rust风格指南
