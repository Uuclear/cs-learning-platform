# 挑战1: 线性查找 —— 仓库盘点

### 难度
⭐

### 描述
你是一家小仓库的管理员。仓库里的货物没有按任何顺序摆放。现在顾客报了一个货号，你需要从货架上找到它并返回它的位置。

实现一个线性查找函数，在货物列表中找到指定货号的位置。

### 输入
- `goods`: 整数列表，表示货架上的货物编号（无序）
- `target`: 整数，要查找的货号

### 输出
- 返回目标货号的索引位置，如果没找到返回-1

### 示例

**示例 1:**
```
输入: goods = [33, 7, 15, 42, 8, 23, 11], target = 42
输出: 3
解释: 货号42在索引3的位置
```

**示例 2:**
```
输入: goods = [33, 7, 15, 42, 8, 23, 11], target = 99
输出: -1
解释: 货架上没有货号99
```

### 约束条件
- 货物列表长度 1 <= n <= 1000
- 货号和目标的取值范围 0 <= x <= 10000

### 提示
- 用for循环遍历列表，逐一对比
- 找到了立即返回索引，不用继续往后找

### 进阶思考
- 如果仓库里可能有多个相同货号的商品，如何找到第一个和最后一个？

### 思路
线性查找的教科书实现。遍历列表，比较每个元素和目标值，相等就返回索引。遍历完还没找到就返回-1。

### 代码
```python
def find_goods(goods, target):
    """
    在仓库货架上找到指定货号的位置
    使用线性查找
    """
    for i in range(len(goods)):
        if goods[i] == target:
            return i
    return -1


# 测试
if __name__ == "__main__":
    goods = [33, 7, 15, 42, 8, 23, 11]

    # 找存在的货号
    result = find_goods(goods, 42)
    print(f"查找42: 索引={result}")  # 输出: 3

    # 找不存在的货号
    result = find_goods(goods, 99)
    print(f"查找99: 索引={result}")  # 输出: -1

    # 找第一个位置的货号
    result = find_goods(goods, 33)
    print(f"查找33: 索引={result}")  # 输出: 0
```

### 复杂度分析
- 时间复杂度: O(n)，最坏需要遍历整个列表
- 空间复杂度: O(1)，只需要一个循环变量
