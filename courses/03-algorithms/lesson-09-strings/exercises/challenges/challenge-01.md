# 编程挑战 1：最长公共子串

## 题目描述

给定两个字符串 `str1` 和 `str2`，找出它们的最长公共子串（连续的相同字符序列）。

**注意**：子串必须是连续的，与子序列不同。

## 示例

```
输入: str1 = "ABABC", str2 = "BABCA"
输出: "BABC" 或 "ABAB"（长度为4）

输入: str1 = "abcdxyz", str2 = "xyzabcd"
输出: "abcd"（长度为4）

输入: str1 = "zxabcdezy", str2 = "yzabcdezx"
输出: "abcdez"（长度为6）
```

## 要求

- 实现函数 `longest_common_substring(str1: str, str2: str) -> str`
- 时间复杂度尽量优化（目标：O(n × m)）
- 空间复杂度尽量优化
- 返回任意一个最长公共子串即可（如果有多个相同长度的）

## 提示

💡 **动态规划思路**：
- 创建二维DP表，`dp[i][j]` 表示以 `str1[i-1]` 和 `str2[j-1]` 结尾的公共子串长度
- 当 `str1[i-1] == str2[j-1]` 时，`dp[i][j] = dp[i-1][j-1] + 1`
- 否则 `dp[i][j] = 0`
- 记录最大长度和对应位置

💡 **空间优化**：
- 注意到每次只用到前一行的数据，可以考虑空间优化到 O(min(n, m))
