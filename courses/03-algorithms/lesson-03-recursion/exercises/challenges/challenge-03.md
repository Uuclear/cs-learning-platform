## 挑战3: 迷宫寻路（递归回溯）

### 难度
⭐⭐⭐⭐

### 描述
用递归回溯算法在一个迷宫中寻找从起点到终点的路径。

迷宫用一个二维列表表示：
- `0` = 通路
- `1` = 墙壁
- `S` = 起点
- `E` = 终点

规则：
- 每次只能上下左右移动一格
- 不能穿过墙壁
- 不能重复走已经走过的格子
- 找到任意一条路径即可（不需要最短路径）

### 输入
- `maze`: 二维列表表示迷宫
- `start`: 起点坐标 `(row, col)`
- `end`: 终点坐标 `(row, col)`

### 输出
- 如果找到路径，返回路径坐标列表 `[(r1,c1), (r2,c2), ...]`
- 如果无解，返回空列表 `[]`

### 示例

**示例 1:**
```
输入:
maze = [
  ['S', 1, 0, 0],
  [ 0,  1, 0, 1],
  [ 0,  0, 0, 0],
  [ 1,  1, 0, 'E']
]
start = (0, 0)
end = (3, 3)

输出:
[(0,0), (1,0), (2,0), (2,1), (2,2), (1,2), (0,2), (0,3), (2,3), (3,3)]
（注：路径可能不唯一，任何有效路径均可）
```

**示例 2:**
```
输入:
maze = [
  ['S', 1],
  [ 1, 'E']
]
start = (0, 0)
end = (1, 1)

输出:
[]
解释: 起点被墙壁包围，无法到达终点
```

### 约束条件
- 迷宫大小 2x2 到 20x20
- 起点和终点保证是通路
- 迷宫中只有一条有效路径或多个有效路径

### 提示
- 基准情况1：当前位置 == 终点 → 找到路径，返回
- 基准情况2：越界 / 撞墙 / 走过 → 此路不通，回溯
- 递归情况：尝试上下左右四个方向
- 用 visited 集合记录走过的格子，避免走回头路
- 回溯的关键：如果某个方向走不通，要"撤销"当前格子的访问记录

### 进阶思考
- 这个算法能找到最短路径吗？如果不能，如何改造它来找最短路径？
- 这个算法和深度优先搜索（DFS）有什么关系？

### 思路
这是经典的回溯算法：从起点出发，尝试向四个方向走。如果某个方向能走到终点就返回路径；如果走不通就回溯（撤销访问记录），尝试其他方向。

### 代码
```python
def solve_maze(maze, start, end):
    """
    递归回溯求解迷宫
    返回: 路径坐标列表，或空列表表示无解
    """
    rows = len(maze)
    cols = len(maze[0])
    visited = set()  # 记录已访问的格子
    
    def backtrack(r, c):
        """
        递归回溯函数
        从 (r, c) 出发，尝试找到终点
        返回: 从当前位置到终点的路径，或空列表
        """
        # 基准情况1：到达终点
        if (r, c) == end:
            return [(r, c)]
        
        # 基准情况2：无效位置
        # 越界 / 撞墙 / 已访问
        if (r < 0 or r >= rows or c < 0 or c >= cols or
                maze[r][c] == 1 or (r, c) in visited):
            return []
        
        # 标记当前格子为已访问
        visited.add((r, c))
        
        # 递归尝试四个方向：下、右、上、左
        directions = [(1, 0), (0, 1), (-1, 0), (0, -1)]
        for dr, dc in directions:
            nr, nc = r + dr, c + dc
            path = backtrack(nr, nc)
            
            # 如果找到路径，把当前位置加入
            if path:
                return [(r, c)] + path
        
        # 四个方向都走不通，回溯
        visited.remove((r, c))
        return []
    
    return backtrack(start[0], start[1])


def print_maze_with_path(maze, path):
    """打印迷宫并标注路径"""
    path_set = set(path)
    for r in range(len(maze)):
        row_str = ""
        for c in range(len(maze[0])):
            if (r, c) in path_set:
                row_str += " * "   # 路径
            elif maze[r][c] == 1:
                row_str += " # "   # 墙壁
            elif maze[r][c] in ('S', 'E'):
                row_str += f" {maze[r][c]} "  # 起点/终点
            else:
                row_str += " . "   # 通路
        print(row_str)


if __name__ == "__main__":
    print("=== 迷宫寻路 ===\n")
    
    maze1 = [
        ['S', 1, 0, 0],
        [0,   1, 0, 1],
        [0,   0, 0, 0],
        [1,   1, 0, 'E']
    ]
    
    print("--- 迷宫1 ---")
    path1 = solve_maze(maze1, (0, 0), (3, 3))
    print_maze_with_path(maze1, path1)
    print(f"\n路径长度: {len(path1)} 步")
    print(f"路径坐标: {path1}")
    # 输出示例:
    #  *  #  *  * 
    #  *  #  *  # 
    #  *  *  *  * 
    #  #  #  *  E 
    #
    # 路径长度: 10 步
    # 路径坐标: [(0, 0), (1, 0), (2, 0), (2, 1), (2, 2), (1, 2), (0, 2), (0, 3), (2, 3), (3, 3)]
    
    print()
    
    print("--- 迷宫2（无解）---")
    maze2 = [
        ['S', 1],
        [1, 'E']
    ]
    path2 = solve_maze(maze2, (0, 0), (1, 1))
    print(f"路径: {path2}")
    print("结论: 起点被墙壁包围，无法到达")
```

### 复杂度分析
- 时间复杂度: O(4^(m*n)) 最坏情况 —— 每个格子有4个方向
- 空间复杂度: O(m*n) —— visited 集合 + 调用栈深度
