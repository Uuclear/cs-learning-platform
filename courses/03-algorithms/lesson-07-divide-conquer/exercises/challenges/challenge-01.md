# 编程挑战1：逆序对计数 ⭐⭐

## 问题描述

给定一个数组，计算其中的逆序对数量。逆序对是指数组中满足 `i < j` 且 `arr[i] > arr[j]` 的元素对。

例如，数组 `[2, 3, 8, 6, 1]` 中有5个逆序对：`(2,1), (3,1), (8,6), (8,1), (6,1)`。

## 要求

- 使用分治算法解决，时间复杂度应为 O(n log n)
- 不能使用暴力解法（O(n²)）
- 实现函数 `count_inversions(arr: List[int]) -> int`

## 提示

可以修改归并排序的合并过程，在合并时统计逆序对的数量。当从右半部分取元素时，说明它比左半部分剩余的所有元素都小，因此会产生相应数量的逆序对。

## 测试用例

```python
assert count_inversions([2, 3, 8, 6, 1]) == 5
assert count_inversions([1, 2, 3, 4, 5]) == 0
assert count_inversions([5, 4, 3, 2, 1]) == 10
```
