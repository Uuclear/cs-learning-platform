# 挑战1：个人简介生成器

## 任务描述

创建一个JavaScript程序，能够收集用户的个人信息并生成个性化的简介。

## 具体要求

1. 声明以下变量：
   - `name`：你的姓名（字符串）
   - `age`：你的年龄（数字）
   - `city`：你所在的城市（字符串）
   - `hobby`：你的爱好（字符串）
   - `isStudent`：是否是在校学生（布尔值）

2. 使用模板字符串创建一个包含所有信息的简介文本

3. 将简介输出到控制台，格式如下：
   ```
   === 个人简介 ===
   姓名：张三
   年龄：25岁
   城市：北京
   爱好：编程、阅读、游泳
   学生身份：是
   ===============
   ```

4. 额外挑战：添加一个计算出生年份的功能（假设当前年份是2026年）

## 提示

- 使用 `const` 声明那些不会改变的变量
- 使用 `let` 声明可能需要计算或修改的变量
- 模板字符串使用反引号（`）包围，变量用 `${variable}` 表示
- 布尔值转换为中文可以使用三元运算符：`isStudent ? "是" : "否"`

## 示例输出

```
=== 个人简介 ===
姓名：李小明
年龄：22岁
城市：上海
爱好：摄影、旅行、音乐
学生身份：是
出生年份：2004年
===============
```
