# 编程挑战 1：实现记忆化函数

## 题目描述

实现一个 `memoize` 函数，它接受一个函数作为参数，并返回一个记忆化版本的函数。记忆化函数会缓存之前的计算结果，当使用相同的参数再次调用时，直接返回缓存的结果而不需要重新计算。

## 要求

1. 使用闭包来保存缓存数据
2. 支持任意数量的参数
3. 正确处理不同类型的参数（包括对象、数组等）
4. 缓存键应该基于参数的值而不是引用

## 示例用法

```javascript
const expensiveFunction = (a, b) => {
  console.log(`计算 ${a} + ${b}`);
  return a + b;
};

const memoized = memoize(expensiveFunction);

console.log(memoized(1, 2)); // 输出: "计算 1 + 2" 然后 3
console.log(memoized(1, 2)); // 输出: 3 (直接从缓存返回，没有"计算"日志)
console.log(memoized(2, 3)); // 输出: "计算 2 + 3" 然后 5
```

## 提示

- 考虑如何将参数数组转换为唯一的缓存键
- 可以使用 `JSON.stringify()` 来序列化参数，但要注意它的限制
- 确保缓存对象不会被外部代码访问或修改

## 扩展挑战（可选）

- 支持自定义缓存键生成函数
- 添加缓存大小限制和LRU淘汰策略
- 支持异步函数的记忆化
