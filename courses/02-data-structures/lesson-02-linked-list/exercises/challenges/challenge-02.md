# 编程挑战 2：链表的中间节点查找 ⭐⭐

## 题目描述
给定一个非空单链表，返回链表的中间节点。

如果链表有两个中间节点（偶数长度），返回第二个中间节点。

## 要求
- 函数签名：`def find_middle_node(head) -> ListNode`
- 使用快慢指针法（Floyd's Algorithm）
- 时间复杂度要求：O(n)
- 空间复杂度要求：O(1)

## 示例
```
输入: 1 -> 2 -> 3 -> 4 -> 5 -> None
输出: 3 -> 4 -> 5 -> None (返回值为3的节点)

输入: 1 -> 2 -> 3 -> 4 -> None  
输出: 3 -> 4 -> None (返回值为3的节点，第二个中间节点)

输入: 1 -> None
输出: 1 -> None
```

## 提示
- 快指针每次走两步，慢指针每次走一步
- 当快指针到达末尾时，慢指针正好在中间
- 注意处理边界条件（空链表、单节点）

> 💡 **思考**：为什么快慢指针能保证找到中间节点？如果要找第一个中间节点（偶数长度时），应该如何修改？
