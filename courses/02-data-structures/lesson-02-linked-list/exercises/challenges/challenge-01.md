# 编程挑战 1：链表的长度计算 ⭐

## 题目描述
实现一个函数，计算给定链表的长度。

## 要求
- 函数签名：`def get_linked_list_length(head) -> int`
- `head` 是链表的头节点（可能是 `None`）
- 返回链表中节点的数量
- 时间复杂度要求：O(n)
- 空间复杂度要求：O(1)

## 示例
```
输入: 1 -> 2 -> 3 -> 4 -> 5 -> None
输出: 5

输入: None (空链表)
输出: 0

输入: 42 -> None
输出: 1
```

## 提示
- 使用一个指针遍历整个链表
- 每访问一个节点，计数器加1
- 当指针为None时停止

> 💡 **思考**：为什么不能像数组一样直接获取长度？这体现了链表的什么特性？
