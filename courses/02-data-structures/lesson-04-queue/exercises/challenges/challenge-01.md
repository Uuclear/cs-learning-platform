# 挑战1: 约瑟夫环问题

## 难度
⭐⭐⭐

## 描述
N个人围成一圈，从第一个人开始报数，数到M的人出圈，然后从下一个人重新开始报数。重复这个过程，直到所有人出圈。求最后出圈的顺序。

这是一个经典的队列应用问题，可以用队列很好地模拟。

## 输入
- `people`: 人数 N（正整数）
- `m`: 报数的数字（正整数）

## 输出
- 返回所有人出圈的顺序列表

## 示例

**示例 1:**
```
输入: people=5, m=3
输出: [3, 1, 5, 2, 4]
解释:
初始: [1, 2, 3, 4, 5]
数1→2, 数2→3, 数3→5 → 3出圈
剩余: [4, 5, 1, 2]
数1→4, 数2→5, 数3→1 → 1出圈
剩余: [2, 4, 5]
数1→2, 数2→4, 数3→5 → 5出圈
...以此类推
```

**示例 2:**
```
输入: people=7, m=2
输出: [2, 4, 6, 1, 5, 3, 7]
```

## 约束条件
- 1 <= people <= 100
- 1 <= m <= 50

## 提示
- 用队列模拟围圈的过程
- 报数不到M的人，出队后重新入队（放到队尾）
- 报到M的人，直接出队（不再入队）
- 重复直到队列为空

### 思路
用队列模拟围圈：把所有人放入队列。每次循环，前 m-1 个人出队后重新入队（模拟报数过程），第 m 个人直接出队（淘汰）。

### 代码
```python
from collections import deque


def josephus(people, m):
    """
    约瑟夫环问题 - 用队列模拟
    参数:
        people: 总人数
        m: 报数到m的人出圈
    返回:
        出圈顺序列表
    """
    # 初始化队列：所有人按顺序入队
    queue = deque(range(1, people + 1))
    result = []

    while queue:
        # 前 m-1 个人出队后重新入队（报数过程）
        for _ in range(m - 1):
            queue.append(queue.popleft())
        # 第 m 个人出队（淘汰）
        result.append(queue.popleft())

    return result


# 测试
if __name__ == "__main__":
    print("=== 约瑟夫环问题 ===\n")

    result1 = josephus(5, 3)
    print(f"people=5, m=3 → 出圈顺序: {result1}")
    # 输出: [3, 1, 5, 2, 4]

    result2 = josephus(7, 2)
    print(f"people=7, m=2 → 出圈顺序: {result2}")
    # 输出: [2, 4, 6, 1, 5, 3, 7]

    result3 = josephus(10, 4)
    print(f"people=10, m=4 → 出圈顺序: {result3}")
    # 输出: [4, 8, 2, 7, 3, 10, 9, 1, 6, 5]
```

### 复杂度分析
- 时间复杂度: O(n * m)，每个人最多入队 m 次
- 空间复杂度: O(n)，队列存储所有人
