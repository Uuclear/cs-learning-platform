# 挑战3：数据流的中位数

## 难度
⭐⭐⭐⭐⭐

## 描述
设计一个数据结构，支持：
1. `addNum(num)`: 从数据流中添加一个数字
2. `findMedian()`: 返回当前所有数字的中位数

要求：每次添加后都能快速查询中位数。

## 中位数定义
- 如果数字个数是奇数，中位数是排序后的中间值
- 如果数字个数是偶数，中位数是排序后中间两个值的平均数

## 示例
```
addNum(1)
addNum(2)
findMedian() -> 1.5  ( (1+2)/2 )
addNum(3)
findMedian() -> 2.0
```

## 提示
- 维护两个堆：一个最大堆存较小的一半，一个最小堆存较大的一半
- 保持两个堆的大小之差不超过1
- 中位数要么是某个堆的堆顶，要么是两个堆顶的平均值

## 思考
- 如果每次都要重新排序找中位数，时间复杂度是多少？
- 双堆方案相比有什么优势？
