# 挑战1：合并K个有序链表

## 难度
⭐⭐⭐

## 描述
给定K个有序链表，将它们合并成一个有序链表。这是LeetCode经典题目，用堆可以优雅地解决。

## 输入
- `lists`: 一个包含K个有序链表的列表

## 输出
- 合并后的一个有序链表

## 示例
```
输入: [1->4->5, 1->3->4, 2->6]
输出: 1->1->2->3->4->4->5->6
```

## 提示
- 每个链表的头节点放入最小堆
- 每次从堆中取出最小节点
- 将取出节点的下一个节点放入堆中
- 重复直到堆为空

- 为什么用堆比用排序更高效？
