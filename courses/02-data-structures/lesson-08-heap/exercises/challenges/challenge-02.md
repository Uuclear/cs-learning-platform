# 挑战2：滑动窗口最大值

## 难度
⭐⭐⭐⭐

## 描述
给定一个数组和一个大小为K的滑动窗口，窗口从左到右移动，每次输出窗口中的最大值。要求时间复杂度优于O(n*k)。

## 输入
- `nums`: 整数数组
- `k`: 滑动窗口大小

## 输出
- 一个数组，包含每个窗口的最大值

## 示例
```
输入: nums = [1, 3, -1, -3, 5, 3, 6, 7], k = 3
输出: [3, 3, 5, 5, 6, 7]
解释:
  窗口位置           最大值
  [1, 3, -1] -3  5  3  6  7    3
   1 [3, -1, -3] 5  3  6  7    3
   1  3 [-1, -3, 5] 3  6  7    5
   1  3  -1 [-3, 5, 3] 6  7    5
   1  3  -1  -3 [5, 3, 6] 7    6
   1  3  -1  -3  5 [3, 6, 7]   7
```

## 提示
- 可以用最大堆存储元素及其索引
- 堆顶元素可能已经不在窗口内，需要检查
- Python的heapq是最小堆，需要存储负数来模拟最大堆
- 注意处理窗口外的过期元素
