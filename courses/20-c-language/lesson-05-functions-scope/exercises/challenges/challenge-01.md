# 挑战 1：创建一个数学工具库

## 目标
创建一个完整的数学工具库，包含多个函数，并正确使用头文件和源文件分离。

## 要求
1. 创建 `math_lib.h` 头文件，包含以下函数声明：
   - `int is_prime(int n)` - 判断是否为质数
   - `int gcd(int a, int b)` - 计算最大公约数
   - `int lcm(int a, int b)` - 计算最小公倍数
   - `double distance_2d(double x1, double y1, double x2, double y2)` - 计算二维距离

2. 创建 `math_lib.c` 源文件，实现上述所有函数

3. 创建 `main.c` 文件，编写测试代码验证所有函数的正确性

4. 使用适当的头文件保护（include guards）

5. 确保代码遵循良好的编程实践（错误处理、注释等）

## 提示
- 质数判断：注意处理边界情况（n <= 1）
- GCD可以使用欧几里得算法的递归或迭代版本
- LCM可以通过公式 `lcm(a,b) = (a * b) / gcd(a,b)` 计算
- 距离计算使用勾股定理：√((x2-x1)² + (y2-y1)²)

## 扩展挑战（可选）
- 添加一个函数 `void print_primes_up_to(int limit)`，打印指定范围内的所有质数
- 实现一个统计函数，记录每个数学函数被调用的次数（使用static变量）

## 评估标准
- ✅ 代码结构正确（头文件/源文件分离）
- ✅ 函数实现正确且高效
- ✅ 包含适当的错误处理
- ✅ 代码注释清晰
- ✅ 测试用例覆盖各种情况
