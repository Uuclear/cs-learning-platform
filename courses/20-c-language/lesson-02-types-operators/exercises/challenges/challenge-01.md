# 挑战1：安全的整数运算库

## 背景
在实际的C语言程序中，整数溢出是一个常见的安全漏洞来源。许多安全问题（如缓冲区溢出）都源于未检测的整数溢出。

## 任务
创建一个简单的安全整数运算库，包含以下函数：

1. `int safe_add(int a, int b, int *result)` - 安全加法
2. `int safe_sub(int a, int b, int *result)` - 安全减法  
3. `int safe_mul(int a, int b, int *result)` - 安全乘法

每个函数都应该：
- 在发生溢出/下溢时返回0（失败）
- 在成功时将结果存储在`result`指针指向的位置，并返回1（成功）

## 要求
- 使用`<limits.h>`中的常量（如`INT_MAX`, `INT_MIN`）
- 考虑正数和负数的各种组合情况
- 编写测试代码验证你的函数在各种边界情况下都能正确工作

## 提示
- 对于加法：检查 `a > INT_MAX - b`（当b为正时）或 `a < INT_MIN - b`（当b为负时）
- 对于减法：可以转换为加法来处理 `a - b = a + (-b)`
- 对于乘法：需要更复杂的检查，考虑符号和绝对值

## 扩展挑战
- 为`unsigned int`类型也实现相应的安全运算函数
- 实现安全的数组索引计算函数
