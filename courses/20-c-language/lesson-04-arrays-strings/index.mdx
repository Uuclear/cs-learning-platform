# 数组与字符串

## 开场白：储物柜的奇妙世界

想象一下，你刚搬进新家，面对一堆杂乱的物品——书籍、衣服、餐具、工具……你需要一个系统来整理它们。于是你买了一个大型储物柜，这个储物柜有多个格子，每个格子都有编号（0号、1号、2号……）。你可以把相同类型的物品放在连续的格子里，比如把所有的书从0号格子开始依次摆放。

在C语言的世界里，**数组**就是这样的储物柜！它是一块连续的内存空间，用来存储相同类型的数据。而**字符串**呢？它其实就是专门用来存放字符的特殊储物柜，只不过这个储物柜有个特殊的规矩：最后一个格子必须放一个特殊的"结束标志"（null字符 `\0`），告诉程序"到这里就结束了"。

今天，我们就一起来探索这个储物柜的奇妙世界，学会如何高效地使用数组和字符串，同时避免那些可能让你的程序"翻车"的陷阱！

## 核心概念：数组的内存布局与字符串的本质

### 一维数组：线性储物柜

一维数组是最基础的数组形式。它的定义语法很简单：

```c
数据类型 数组名[元素个数];
```

比如：
```c
int scores[5];        // 声明一个可以存放5个整数的数组
char name[20];        // 声明一个可以存放20个字符的数组
```

**内存布局的秘密**：数组在内存中是连续存储的。假设 `int scores[5]` 的起始地址是 1000，那么：
- `scores[0]` 在地址 1000
- `scores[1]` 在地址 1004（假设int占4字节）
- `scores[2]` 在地址 1008
- ...以此类推

这种连续的内存布局让数组的访问速度非常快，因为CPU可以通过简单的计算直接跳转到任意位置。

### 二维数组：储物柜中的储物柜

二维数组就像是储物柜中的储物柜，或者说是表格形式的数据结构：

```c
int matrix[3][4];     // 3行4列的整数矩阵
```

在内存中，二维数组实际上是按行优先的方式连续存储的：
```
matrix[0][0], matrix[0][1], matrix[0][2], matrix[0][3],
matrix[1][0], matrix[1][1], matrix[1][2], matrix[1][3],
matrix[2][0], matrix[2][1], matrix[2][2], matrix[2][3]
```

### 字符串的本质：以null结尾的字符数组

这是C语言中最重要也最容易被误解的概念之一！**C语言中没有真正的"字符串"类型**，所谓的字符串其实就是 `char` 类型的数组，只不过这个数组的最后一个元素必须是 `\0`（ASCII值为0的字符）。

```c
// 这两种声明方式是等价的！
char greeting[] = "Hello";
char greeting[] = {'H', 'e', 'l', 'l', 'o', '\0'};
```

注意：当你用字符串字面量初始化字符数组时，编译器会自动在末尾添加 `\0`。但是如果你手动逐个赋值字符，就必须自己记得添加 `\0`！

```c
// 错误示例 - 缺少结束符
char bad_string[5] = {'H', 'e', 'l', 'l', 'o'};  // 没有 \0！

// 正确示例
char good_string[6] = {'H', 'e', 'l', 'l', 'o', '\0'};  // 有 \0！
```

## 代码演示：数组遍历和字符串处理

让我们通过实际代码来看看数组和字符串的基本操作。

### 数组的基本操作

数组的常见操作包括初始化、遍历、查找和排序。下面是一个完整的例子：

```c
#include <stdio.h>

int main() {
    // 初始化数组
    int numbers[] = {10, 20, 30, 40, 50};
    int size = sizeof(numbers) / sizeof(numbers[0]);
    
    printf("数组内容：");
    // 遍历数组
    for (int i = 0; i < size; i++) {
        printf("%d ", numbers[i]);
    }
    printf("\n");
    
    // 查找最大值
    int max = numbers[0];
    for (int i = 1; i < size; i++) {
        if (numbers[i] > max) {
            max = numbers[i];
        }
    }
    printf("最大值：%d\n", max);
    
    return 0;
}
```

### 字符串的基本操作

字符串的操作更加丰富，包括复制、连接、比较等：

```c
#include <stdio.h>
#include <string.h>

int main() {
    char source[] = "Hello";
    char destination[20];
    
    // 复制字符串
    strcpy(destination, source);
    printf("复制后：%s\n", destination);
    
    // 连接字符串
    strcat(destination, " World");
    printf("连接后：%s\n", destination);
    
    // 计算长度
    printf("字符串长度：%lu\n", strlen(destination));
    
    // 比较字符串
    char another[] = "Hello World";
    if (strcmp(destination, another) == 0) {
        printf("两个字符串相等！\n");
    }
    
    return 0;
}
```

## 深入理解：字符串函数实现原理

理解标准库函数的内部实现有助于我们更好地使用它们，也能帮助我们在需要时自己实现类似功能。

### strlen 的实现

`strlen` 函数计算字符串的长度（不包括 `\0`）：

```c
size_t my_strlen(const char *str) {
    size_t length = 0;
    while (*str != '\0') {
        length++;
        str++;
    }
    return length;
}
```

### strcpy 的实现

`strcpy` 函数将源字符串复制到目标字符串：

```c
char* my_strcpy(char *dest, const char *src) {
    char *original_dest = dest;
    while (*src != '\0') {
        *dest = *src;
        dest++;
        src++;
    }
    *dest = '\0';  // 不要忘记复制结束符！
    return original_dest;
}
```

### strcat 的实现

`strcat` 函数将源字符串连接到目标字符串的末尾：

```c
char* my_strcat(char *dest, const char *src) {
    char *original_dest = dest;
    
    // 先找到目标字符串的末尾
    while (*dest != '\0') {
        dest++;
    }
    
    // 然后复制源字符串
    while (*src != '\0') {
        *dest = *src;
        dest++;
        src++;
    }
    *dest = '\0';
    return original_dest;
}
```

## 实际应用：文本统计与字符串反转

让我们看看数组和字符串在实际项目中的应用。

### 文本字符统计

统计一段文本中各种字符的出现次数：

```c
#include <stdio.h>
#include <string.h>
#include <ctype.h>

void count_characters(const char *text) {
    int letters = 0, digits = 0, spaces = 0, others = 0;
    
    for (int i = 0; text[i] != '\0'; i++) {
        if (isalpha(text[i])) {
            letters++;
        } else if (isdigit(text[i])) {
            digits++;
        } else if (isspace(text[i])) {
            spaces++;
        } else {
            others++;
        }
    }
    
    printf("字母：%d\n", letters);
    printf("数字：%d\n", digits);
    printf("空格：%d\n", spaces);
    printf("其他：%d\n", others);
}

int main() {
    char text[] = "Hello World! 123";
    count_characters(text);
    return 0;
}
```

### 字符串反转

将字符串中的字符顺序反转：

```c
#include <stdio.h>
#include <string.h>

void reverse_string(char *str) {
    int length = strlen(str);
    for (int i = 0; i < length / 2; i++) {
        char temp = str[i];
        str[i] = str[length - 1 - i];
        str[length - 1 - i] = temp;
    }
}

int main() {
    char text[] = "Hello World";
    printf("原字符串：%s\n", text);
    reverse_string(text);
    printf("反转后：%s\n", text);
    return 0;
}
```

## 常见误区：缓冲区溢出与安全编码

### 缓冲区溢出的危险

缓冲区溢出是C语言中最常见的安全漏洞之一。当向数组或字符串写入超出其容量的数据时，就会发生缓冲区溢出：

```c
// 危险示例！
char buffer[10];
strcpy(buffer, "This is a very long string that will overflow!");  // 溢出！
```

这种溢出会覆盖相邻的内存区域，可能导致：
- 程序崩溃
- 数据损坏
- 安全漏洞（攻击者可能利用此执行恶意代码）

### 安全编码实践

为了避免缓冲区溢出，我们应该：

1. **使用安全的字符串函数**：
   - `strncpy` 代替 `strcpy`
   - `strncat` 代替 `strcat`
   - `snprintf` 代替 `sprintf`

2. **始终检查边界**：
```c
char dest[20];
const char *src = "Hello World";
if (strlen(src) < sizeof(dest)) {
    strcpy(dest, src);
} else {
    // 处理错误情况
    printf("源字符串太长！\n");
}
```

3. **使用现代编译器的安全检查**：
```c
// GCC/Clang 可以启用栈保护

```

## 小结与思考

今天我们学习了C语言中数组和字符串的核心概念：

- **数组**是连续的内存块，用于存储相同类型的数据
- **字符串**本质上是以 `\0` 结尾的字符数组
- 标准库提供了丰富的字符串操作函数（strcpy, strcat, strlen, strcmp等）
- **缓冲区溢出**是最危险的常见错误，必须时刻警惕
- 安全编码实践包括使用安全函数、边界检查和现代编译器特性
