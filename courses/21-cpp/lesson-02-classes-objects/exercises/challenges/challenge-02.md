# 挑战2：汽车租赁系统

## 背景
你正在为一家汽车租赁公司开发一个简单的车辆管理系统。需要管理不同类型的车辆和租赁记录。

## 要求
创建两个类：`Car` 和 `RentalSystem`，满足以下要求：

### Car 类要求

#### 基本属性（私有成员）
- 车牌号（string）
- 品牌（string）
- 型号（string）
- 日租金（double）
- 是否可用（bool）
- 当前租客（string，可为空）

#### 功能要求（公共接口）
1. **构造函数**：初始化车辆信息，初始状态为可用
2. **获取车辆信息**：返回完整的车辆信息
3. **检查可用性**：返回车辆是否可以租赁
4. **租赁车辆**：标记车辆为已租出，并记录租客姓名
5. **归还车辆**：标记车辆为可用，并清除租客信息
6. **计算租赁费用**：根据租赁天数计算总费用
7. **显示车辆状态**：显示车辆的当前状态（可用/已租出）

### RentalSystem 类要求

#### 基本属性（私有成员）
- 车辆列表（vector<Car>）
- 租赁记录（可以简单记录）

#### 功能要求（公共接口）
1. **添加车辆**：向系统中添加新车
2. **查找可用车辆**：返回所有可用的车辆
3. **租赁车辆**：根据车牌号租赁指定车辆
4. **归还车辆**：根据车牌号归还指定车辆
5. **显示所有车辆**：显示系统中所有车辆的状态
6. **获取总收入**：计算系统至今的总收入

### 额外要求
- 使用适当的封装原则，保护内部数据
- 所有查询函数声明为 const 成员函数
- 处理边界情况（如租赁不存在的车辆、租赁已租出的车辆等）
- 在 RentalSystem 中实现基本的错误处理

## 测试用例
你的系统应该能够通过以下测试场景：

```cpp
// 创建租赁系统
RentalSystem rentalSys;

// 添加车辆
rentalSys.addCar("京A12345", "Toyota", "Camry", 300.0);
rentalSys.addCar("京B67890", "Honda", "Accord", 280.0);

// 显示所有车辆
rentalSys.displayAllCars();

// 租赁车辆
rentalSys.rentCar("京A12345", "张三", 5); // 租赁5天

// 查看可用车辆
auto availableCars = rentalSys.getAvailableCars();
cout << "可用车辆数量: " << availableCars.size() << endl;

// 归还车辆
rentalSys.returnCar("京A12345");

// 显示总收入
cout << "总收入: " << rentalSys.getTotalRevenue() << "元" << endl;
```

## 进阶挑战（可选）
- 实现按品牌或型号搜索车辆的功能
- 添加租赁历史记录功能
- 实现会员折扣系统（不同会员等级有不同的折扣）
- 添加车辆维护状态（维修中的车辆不可租赁）

## 提示
- 考虑使用 STL 容器来管理车辆列表
- 在查找车辆时，考虑使用车牌号作为唯一标识
- 合理设计类之间的交互关系
- 注意 const 正确性和异常安全
