# 挑战1：学生管理系统

## 背景
你是一家教育机构的开发人员，需要创建一个学生管理系统来管理学生信息和成绩。

## 要求
创建一个 `Student` 类，满足以下要求：

### 基本属性（私有成员）
- 学号（string）
- 姓名（string）
- 年龄（int）
- 专业（string）
- 成绩列表（vector<double>）

### 功能要求（公共接口）
1. **构造函数**：能够初始化学生的基本信息
2. **添加成绩**：可以添加单个成绩（0-100分之间）
3. **批量添加成绩**：可以一次性添加多个成绩
4. **计算平均分**：返回所有成绩的平均值
5. **获取最高分**：返回成绩中的最高分
6. **获取最低分**：返回成绩中的最低分
7. **判断是否及格**：如果平均分 >= 60 返回 true
8. **显示学生信息**：显示学生的完整信息，包括基本信息和成绩统计
9. **成绩等级**：根据平均分返回等级（A: >=90, B: >=80, C: >=70, D: >=60, F: <60）

### 额外要求
- 使用适当的访问控制（封装原则）
- 所有查询函数应该声明为 const 成员函数
- 在添加成绩时进行有效性验证（0-100分）
- 考虑边界情况（如没有成绩时的处理）

## 测试用例
你的类应该能够通过以下测试场景：

```cpp
// 创建学生对象
Student student("2023001", "张三", 20, "计算机科学");

// 添加成绩
student.addGrade(85.5);
student.addGrade(92.0);
student.addGrades({78.5, 88.0, 95.5});

// 显示信息
student.displayInfo();

// 验证功能
cout << "平均分: " << student.getAverageGrade() << endl;
cout << "成绩等级: " << student.getGradeLevel() << endl;
cout << "是否及格: " << (student.isPassing() ? "是" : "否") << endl;
```

## 提示
- 考虑使用 STL 容器（如 vector）来存储成绩
- 注意处理空成绩列表的情况
- 合理使用 const 正确性
- 考虑异常情况的处理（如无效年龄、负数成绩等）
