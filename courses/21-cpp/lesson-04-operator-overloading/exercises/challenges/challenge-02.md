# 挑战 2：实现矩阵类的运算符重载

## 目标
实现一个 `Matrix` 类，支持矩阵的基本运算和索引访问。

## 要求
1. **构造函数**：
   - 默认构造（创建1x1零矩阵）
   - 指定行列数的构造函数
   - 从二维数组初始化的构造函数
2. **核心功能**：
   - 获取和设置矩阵元素（使用`operator()`而不是`operator[]`）
   - 获取行数和列数
   - 矩阵转置函数
3. **运算符重载**：
   - 算术运算符：`+`, `-`（同维度矩阵）
   - 标量运算符：`*`, `/`（矩阵与标量的乘除）
   - 复合赋值运算符：`+=`, `-=`, `*=`, `/=`
   - 比较运算符：`==`, `!=`
   - 流运算符：`<<`（格式化输出矩阵）
   - 函数调用运算符：`operator()`用于二维索引访问
4. **错误处理**：
   - 维度不匹配时抛出异常
   - 索引越界时抛出异常
   - 无效的矩阵操作抛出异常

## 示例用法
```cpp
Matrix m1(2, 3);  // 2x3 矩阵
m1(0, 0) = 1; m1(0, 1) = 2; m1(0, 2) = 3;
m1(1, 0) = 4; m1(1, 1) = 5; m1(1, 2) = 6;

Matrix m2 = m1 + m1;  // 矩阵加法
Matrix m3 = m1 * 2.0; // 标量乘法

cout << "Original matrix:" << endl;
cout << m1 << endl;

cout << "Doubled matrix:" << endl;
cout << m3 << endl;

// 转置
Matrix m4 = m1.transpose();
cout << "Transposed matrix:" << endl;
cout << m4 << endl;
```

## 输出格式要求
矩阵输出应该每行一个，元素之间用空格分隔：
```
1 2 3
4 5 6
```

## 提示
- 使用动态内存分配或`std::vector<std::vector<double>>`
- `operator()`比`operator[]`更适合二维索引，因为可以接受两个参数
- 考虑深拷贝语义（拷贝构造函数和赋值运算符）
- 对于大型矩阵，考虑性能优化（如避免不必要的拷贝）
