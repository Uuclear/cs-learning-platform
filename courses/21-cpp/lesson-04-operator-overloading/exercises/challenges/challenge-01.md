# 挑战 1：实现一个分数类

## 目标
实现一个 `Fraction` 类，支持分数的算术运算和比较。

## 要求
1. **构造函数**：支持默认构造（0/1）、单参数构造（整数转分数）和双参数构造（分子/分母）
2. **简化分数**：在构造时自动将分数化简为最简形式（使用最大公约数）
3. **运算符重载**：
   - 算术运算符：`+`, `-`, `*`, `/`
   - 复合赋值运算符：`+=`, `-=`, `*=`, `/=`
   - 比较运算符：`==`, `!=`, `<`, `<=`, `>`, `>=`
   - 流运算符：`<<`（输出格式如 "3/4" 或 "2" 如果分母为1）
4. **特殊处理**：
   - 分母不能为0，需要抛出异常
   - 负号应该统一放在分子上
   - 支持与整数的混合运算（如 `fraction + 5` 和 `5 + fraction`）

## 示例用法
```cpp
Fraction f1(6, 8);  // 自动化简为 3/4
Fraction f2(1, 3);
Fraction f3 = f1 + f2;  // 结果应该是 13/12
cout << f3 << endl;     // 输出 "13/12"

Fraction f4(2, 1);
cout << f4 << endl;     // 输出 "2"

bool result = (f1 > f2); // true
```

## 提示
- 使用欧几里得算法计算最大公约数（GCD）
- 注意处理负数的情况
- 考虑使用友元函数来处理左操作数为整数的情况
- 记得重载所有相关的运算符以保持一致性
