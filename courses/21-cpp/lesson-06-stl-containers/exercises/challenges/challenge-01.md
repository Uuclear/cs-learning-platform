# 挑战1：图书管理系统

## 背景
你是一家小型图书馆的管理员，需要开发一个简单的图书管理系统来跟踪图书借阅情况。

## 要求
使用STL容器实现以下功能：

### 数据结构设计
- 使用 `std::map<std::string, Book>` 存储图书信息，其中键是ISBN号，值是Book结构体
- Book结构体包含：书名(title)、作者(author)、出版年份(year)、是否可借出(available)
- 使用 `std::map<std::string, std::vector<std::string>>` 跟踪借阅记录，键是读者姓名，值是该读者借阅的ISBN列表

### 功能实现
1. **添加图书**：输入ISBN、书名、作者、年份，将图书添加到系统中
2. **借书**：输入读者姓名和ISBN，如果图书可借出，则标记为不可借出，并添加到读者借阅记录
3. **还书**：输入读者姓名和ISBN，将图书标记为可借出，并从读者借阅记录中移除
4. **查询图书**：输入ISBN，显示图书详细信息和当前状态
5. **查询读者**：输入读者姓名，显示该读者当前借阅的所有图书
6. **统计功能**：
   - 显示所有可借出的图书（按书名字母顺序）
   - 显示最受欢迎的图书（被借阅次数最多的前3本）

### 额外挑战（选做）
- 实现图书搜索功能：根据书名或作者模糊匹配
- 添加借阅期限功能：记录借阅日期，自动计算逾期天数

## 提示
- 考虑使用结构化绑定（C++17）简化代码
- 注意处理边界情况：图书不存在、读者不存在、重复借阅等
- 使用const引用避免不必要的拷贝

## 测试用例
```cpp
// 示例测试数据
Library lib;
lib.addBook("978-0134685991", "Effective Modern C++", "Scott Meyers", 2014);
lib.addBook("978-0321563842", "The C++ Programming Language", "Bjarne Stroustrup", 2009);

lib.borrowBook("张三", "978-0134685991");
lib.borrowBook("李四", "978-0321563842");

lib.displayAvailableBooks(); // 应该显示空列表
lib.returnBook("张三", "978-0134685991");
lib.displayAvailableBooks(); // 应该显示Effective Modern C++
```
