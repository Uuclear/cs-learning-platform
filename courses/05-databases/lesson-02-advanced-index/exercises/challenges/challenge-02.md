# 实现B+树索引模拟器

## 难度：⭐⭐⭐⭐

## 描述

B+树是现代数据库系统中最常用的索引数据结构。你的任务是实现一个完整的B+树模拟器，能够演示B+树的插入、删除、搜索和范围查询操作，并可视化其内部结构变化。

这个模拟器将帮助你深入理解B+树的工作原理，以及为什么它如此适合用作数据库索引。

## 功能要求

### 基本功能
1. **B+树节点实现**：
   - 支持内部节点和叶子节点
   - 内部节点存储键值和子节点指针
   - 叶子节点存储键值和实际数据，并维护链表指针

2. **核心操作**：
   - `insert(key, value)`: 插入键值对
   - `search(key)`: 搜索键对应的值
   - `delete(key)`: 删除指定键（可选但推荐）
   - `range_query(start_key, end_key)`: 范围查询

3. **树结构维护**：
   - 节点分裂（当节点满时）
   - 节点合并（当节点过小时，删除操作时）
   - 保持树的平衡性

### 高级功能
4. **可视化功能**：
   - 打印树的层次结构
   - 显示叶子节点链表
   - 展示每次操作后的树状态变化

5. **性能分析**：
   - 记录树的高度
   - 统计节点访问次数
   - 比较与普通二叉搜索树的性能差异

6. **交互式演示**：
   - 支持命令行交互
   - 允许用户逐步执行操作并观察变化

## 技术要求

- 使用Python 3.10+实现
- 不使用第三方库（仅使用标准库）
- 代码要有完整的中文注释
- 遵循PEP 8编码规范
- 包含完整的单元测试

## 测试用例

你的实现应该能正确处理以下场景：

1. **基本插入和搜索**：
   ```python
   tree = BPlusTree(order=4)
   tree.insert(10, "data10")
   tree.insert(20, "data20")
   assert tree.search(10) == "data10"
   ```

2. **节点分裂**：
   - 插入足够多的数据触发根节点分裂
   - 验证树的高度正确增加

3. **范围查询**：
   ```python
   results = tree.range_query(15, 25)
   # 应该返回键在[15, 25]范围内的所有数据
   ```

4. **边界情况**：
   - 空树的搜索和范围查询
   - 重复键的处理（覆盖或报错）
   - 不同数据类型的键（整数、字符串）

## 提示

- B+树的阶数（order）决定了每个节点最多包含多少个键
- 叶子节点应该形成双向链表，便于范围查询
- 内部节点只存储键值，不存储实际数据
- 分裂时，中间键要提升到父节点
- 考虑递归和迭代两种实现方式的优缺点
