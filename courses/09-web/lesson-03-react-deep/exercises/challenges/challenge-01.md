# 挑战 1: 实现自定义的 React.memo 高阶组件

⭐⭐⭐ 难度 | 预计完成时间: 15分钟

## 背景
在实际开发中，我们经常需要对组件进行性能优化。React.memo 是一个非常有用的高阶组件，但它只进行浅比较。有时我们需要更灵活的比较逻辑。

## 任务
实现一个自定义的 `customMemo` 高阶组件，它接受两个参数：
1. 组件函数
2. 自定义比较函数 `(prevProps, nextProps) => boolean`

如果自定义比较函数返回 `true`，表示 props 没有变化，应该跳过渲染；返回 `false` 表示需要重新渲染。

## 要求
- 使用 Python 类模拟 React 组件
- 实现完整的 `customMemo` 函数
- 提供一个使用示例，展示如何用深度比较来优化渲染
- 包含中文注释说明关键逻辑

## 提示
- 参考 `example-02-render-optimization.py` 中的 `react_memo` 实现
- 考虑如何存储和比较前一次的 props
- 深度比较可以使用递归或 JSON 序列化（简化版）

## 验证
你的实现应该能够：
1. 正确缓存组件实例
2. 根据自定义比较函数决定是否渲染
3. 处理复杂的嵌套对象 props
