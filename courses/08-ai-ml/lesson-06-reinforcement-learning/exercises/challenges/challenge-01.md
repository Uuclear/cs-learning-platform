# 实现Q-learning解决迷宫问题 - ⭐⭐⭐

## 任务描述

在这个挑战中，你需要实现一个完整的Q-learning算法来解决自定义的迷宫问题。你将创建一个网格世界环境，并训练智能体找到从起点到目标的最优路径。

## 具体要求

### 1. 环境设计
- 创建一个8x8的网格世界
- 设置起始位置在左上角(0,0)，目标位置在右下角(7,7)
- 在网格中随机放置10个障碍物（不能放在起点和终点）
- 智能体每次移动获得-0.1的奖励，碰撞障碍物获得-1.0奖励，到达目标获得+10.0奖励

### 2. Q-learning实现
- 实现完整的Q-learning算法
- 使用ε-greedy策略进行动作选择
- 支持超参数调整（学习率、折扣因子、epsilon值）
- 实现状态离散化（每个网格位置作为一个状态）

### 3. 训练和评估
- 训练智能体至少500个episode
- 在训练过程中记录每个episode的总奖励和步数
- 训练完成后，使用学到的策略演示从起点到终点的路径
- 分析不同超参数对学习效果的影响

### 4. 额外挑战（可选）
- 实现经验回放机制，提高样本效率
- 添加可视化功能，动态显示智能体的学习过程
- 尝试不同的探索策略（如softmax、UCB）并比较效果

## 提交内容

1. 完整的Python代码文件
2. 训练过程的输出日志
3. 最终学到的策略演示结果
4. 对超参数影响的简要分析报告

## 评判标准

- **正确性**：代码能够正确运行并解决迷宫问题
- **完整性**：满足所有基本要求
- **优化**：代码结构清晰，有适当的注释和错误处理
- **分析深度**：对结果有深入的理解和分析

- 记住要重置环境状态在每个episode开始时
