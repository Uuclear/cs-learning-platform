# 编程挑战1: 密码强度检查器 ⭐

## 背景
弱密码是网络安全的最大威胁之一。据统计，超过80%的数据泄露事件都与弱密码或重复使用密码有关。

## 任务
实现一个密码强度检查器，要求能够：
- 检查密码长度是否至少8位
- 检查是否包含大写字母、小写字母、数字和特殊字符（!@#$%^&*(),.?":{}|<>）
- 返回密码强度评分（0-4分，每满足一个条件得1分）

## 要求
- 使用Python 3.10+
- 函数名为 `check_password_strength(password: str) -> int`
- 添加适当的错误处理
- 提供完整的文档字符串说明函数功能

## 示例
```python
print(check_password_strength("123456"))        # 输出: 1
print(check_password_strength("Password123"))   # 输出: 3  
print(check_password_strength("MyP@ssw0rd!"))   # 输出: 4
```

## 提示
- 可以使用正则表达式来检查字符类型
- 考虑使用 `re.search()` 函数
- 特殊字符集合可以定义为常量
