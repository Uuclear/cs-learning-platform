# 编程挑战 1：WebSocket回显服务器 ⭐

## 题目描述
创建一个简单的WebSocket服务器，它能够接收客户端发送的任何消息，并将相同的消息原样返回给发送者。

## 具体要求
- 使用Python和websockets库实现
- 服务器监听在`ws://localhost:8767`
- 支持多个客户端同时连接
- 每个客户端发送的消息只返回给该客户端（不是广播）
- 在控制台输出收到的消息内容

## 提示
- 可以参考示例代码中的基本服务器结构
- 不需要处理复杂的JSON消息，直接回显字符串即可
- 记得处理连接关闭的情况

## 预期效果
```
客户端发送: "Hello WebSocket!"
服务器收到并回显: "Hello WebSocket!"

客户端发送: "这是一个测试"
服务器收到并回显: "这是一个测试"
```
