# 挑战1：数论变换(NTT)实现

## 背景
FFT使用浮点运算，在处理大整数乘法时可能存在精度问题。数论变换(Number Theoretic Transform, NTT)使用模运算替代复数运算，可以得到精确的整数结果。

## 要求
1. 实现NTT算法，使用合适的质数模数（如998244353）
2. 实现原根的计算和单位根的模幂运算
3. 实现NTT版本的多项式乘法
4. 处理任意长度的输入（通过补零到2的幂）
5. 与FFT版本的结果进行对比验证

## 输入格式
- poly1, poly2: 整数系数列表
- mod: 质数模数（默认998244353）

## 输出格式
- 结果多项式的系数列表（模mod）

## 提示
- 需要找到模mod下的原根
- 单位根为原根的(mod-1)/n次幂
- 逆NTT需要计算模逆元
- 注意处理负数的模运算
