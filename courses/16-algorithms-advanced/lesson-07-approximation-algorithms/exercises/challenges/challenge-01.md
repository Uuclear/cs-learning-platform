# 挑战1：实现背包问题的FPTAS

## 背景
背包问题是经典的NP难优化问题，但它有一个FPTAS（完全多项式时间近似方案），这意味着我们可以得到任意精度的近似解。

## 要求
1. 实现背包问题的标准动态规划解法（基于价值的DP）
2. 实现背包问题的FPTAS版本
3. 比较不同ε值下的精度和运行时间
4. 验证FPTAS的理论保证

## 算法思路
FPTAS的核心思想是：
1. 找到最大价值 vmax = max{vi}
2. 设置缩放因子 K = (ε × vmax) / n
3. 将所有价值 vi' = floor(vi / K)
4. 对缩放后的价值运行标准DP
5. 返回对应的原始价值解

- 测量运行时间并验证时间复杂度 O(n³/ε)
