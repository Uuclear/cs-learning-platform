# 挑战2：近似算法在实际问题中的应用

## 背景
近似算法不仅在理论上有重要意义，在实际应用中也非常有价值。选择一个实际问题，设计并实现相应的近似算法。

## 可选问题（选择一个）

### 选项A：设施选址问题
- **问题描述**：给定客户位置和候选设施位置，选择k个设施位置以最小化所有客户的最大距离
- **近似算法**：实现贪心k-center算法（2-近似）
- **应用场景**：服务器部署、医院选址、仓库布局

### 选项B：作业调度问题
- **问题描述**：将n个作业分配给m台机器，最小化最大完成时间（makespan）
- **近似算法**：实现列表调度算法（2-近似）或LPT算法（4/3-近似）
- **应用场景**：云计算资源分配、生产线调度

### 选项C：最大割问题
- **问题描述**：将图的顶点分为两组，最大化两组之间的边数
- **近似算法**：实现随机化0.5-近似算法或Goemans-Williamson算法（0.878-近似）
- **应用场景**：社区发现、图像分割、VLSI设计

## 要求
1. 详细描述所选问题的数学模型
2. 实现相应的近似算法
3. 分析算法的近似比和时间复杂度
4. 在实际数据集上测试算法性能
5. 与基线方法（如随机算法）进行比较

## 提交内容
- 完整的代码实现
- 算法分析报告（包括近似比证明概要）
- 实验结果和性能分析
- 实际应用场景讨论（500字以上）

## 评分标准
- 问题理解和建模（20分）
- 算法正确实现（30分）
- 理论分析深度（25分）
- 实验设计和结果（15分）
- 应用价值讨论（10分）
