# 挑战1：复杂电路分析

## 问题描述

你遇到了一个复杂的电路，需要使用欧姆定律、KVL和KCL来分析。电路图如下：

```
    +15V
      |
     [R1=5Ω]
      |
     /---[R2=10Ω]---+---[R4=8Ω]---\
    |               |              |
   [R3=6Ω]         [V2=6V]       [R5=4Ω]
    |               |              |
     \--------------+--------------/
                    |
                   GND
```

其中V2是一个6V的电压源，正极朝上。

## 任务要求

1. **识别所有独立回路**并为每个回路设定回路电流方向
2. **应用KVL**为每个回路建立方程
3. **求解回路电流**
4. **计算每个电阻的电流和电压降**
5. **验证KCL**在中间节点（R2、V2、R4连接点）是否成立
6. **计算整个电路消耗的总功率**

## 提示

- 使用网孔电流法（Mesh Current Method）
- 注意电压源V2的方向
- 在建立KVL方程时，要正确处理共享电阻的影响
- 验证你的结果是否满足能量守恒（电源提供的功率等于电阻消耗的功率）

## 评估标准

- 方程建立的正确性
- 计算结果的准确性
- KCL验证的完整性
- 功率平衡的验证
