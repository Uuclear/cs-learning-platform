# 挑战2：电路故障诊断

## 问题描述

你是一名电子工程师，接到一个紧急任务：诊断一块印刷电路板（PCB）的故障。这块电路板是一个简单的LED驱动电路，但LED完全不亮。

电路原理图如下：

```
    +9V (电池)
      |
     [R_limit=330Ω]
      |
     LED (正向压降约2.1V)
      |
     [R_sense=10Ω]
      |
     GND
```

你有以下测量工具：
- 数字万用表（可以测量电压、电流、电阻）
- 示波器（可以观察电压波形）

## 测量结果

你进行了以下测量：

1. **电源电压**：+9V（正常）
2. **R_limit两端电压**：0V
3. **LED两端电压**：0V  
4. **R_sense两端电压**：0V
5. **电路总电流**：0A

## 任务要求

1. **分析可能的故障原因**（至少列出3种可能性）
2. **设计诊断步骤**来确定具体的故障点
3. **使用KVL和欧姆定律**解释为什么所有电压读数都是0V
4. **编写Python脚本**模拟这个故障情况，并与正常工作情况进行对比
5. **提出预防措施**避免类似故障在未来发生

## 提示

- 考虑开路、短路、元件损坏等不同故障模式
- 使用KVL分析：如果电流为0，那么所有电阻上的电压降都应该是0
- 故障可能出现在任何位置：电源连接、电阻、LED、接地等
- 在编写Python脚本时，考虑如何模拟开路情况（无穷大电阻）

## 评估标准

- 故障分析的全面性
- 诊断步骤的逻辑性和可操作性
- 理论分析的正确性
- Python脚本的完整性和实用性
- 预防措施的合理性
