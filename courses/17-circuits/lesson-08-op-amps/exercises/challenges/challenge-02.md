# 挑战 2: 设计温度监控与报警系统

## 背景
你需要设计一个基于运放的温度监控系统，使用 LM35 温度传感器（输出 10mV/°C）来监测环境温度，并在温度超过预设阈值时触发报警。

## 要求
1. **信号调理电路**：LM35 输出范围为 0-1V（对应 0-100°C），但你的比较器参考电压需要可调（20-40°C 范围）
2. **可调阈值电路**：设计一个电位器电路，允许用户设置报警阈值（20°C、25°C、30°C、35°C、40°C）
3. **比较器设计**：选择合适的运放配置实现温度比较功能
4. **输出指示**：比较器输出应能直接驱动 LED 指示灯
5. **抗干扰设计**：考虑加入迟滞（hysteresis）以避免温度在阈值附近波动时产生误触发

## 具体任务
1. 绘制完整的电路原理图（可以手绘或使用电路仿真软件）
2. 计算所有电阻值，确保阈值设置准确
3. 设计迟滞电路，迟滞宽度约为 2°C
4. 编写 Python 仿真代码，模拟不同温度下的系统响应
5. 分析系统的精度和可能的误差来源

## 参数说明
- 电源电压：+5V 单电源
- LM35 输出：10mV/°C
- 报警阈值范围：20-40°C（对应 200-400mV）
- 迟滞要求：±1°C（对应 ±10mV）

## 评估标准
- 电路设计的完整性和实用性
- 参数计算的准确性
- 迟滞电路设计的合理性
- 仿真验证的充分性
- 对实际应用中问题的考虑
