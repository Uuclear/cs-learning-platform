# 挑战 2：多级音频放大器设计与失真分析

## 背景
单级放大器往往无法满足实际应用的增益要求。你需要设计一个两级音频放大器，用于驱动小型扬声器（8Ω 负载）。

## 设计规格
- **总电压增益**：100 倍（40dB）
- **输入信号**：来自麦克风，约 5mV RMS
- **输出功率**：至少 100mW 到 8Ω 负载
- **频率响应**：100Hz - 5kHz（语音频段）
- **总谐波失真（THD）**：< 5% 在最大输出时
- **电源**：12V DC

## 任务要求

### 第一级：电压放大级
1. 设计共射极放大电路，提供主要电压增益（约 30 倍）
2. 输入阻抗 > 1kΩ 以匹配麦克风
3. 使用发射极退化电阻提高线性度
4. 计算并优化静态工作点

### 第二级：功率输出级
1. 设计射极跟随器（共集电极）作为缓冲级
2. 输出阻抗 < 100Ω 以驱动 8Ω 负载
3. 能够提供足够的电流（至少 110mA 峰值）
4. 考虑使用推挽输出级减少交越失真

### 级间耦合
1. 设计合适的耦合电容，确保低频响应
2. 考虑直接耦合的可能性和挑战

### 失真分析与优化
1. 使用 Python 模拟整个两级放大器
2. 分析不同输入幅度下的 THD
3. 识别主要失真来源（截止失真、饱和失真、交越失真）
4. 提出改进方案降低失真

### 实际考虑
1. **功耗分析**：计算静态功耗和最大动态功耗
2. **热设计**：评估是否需要散热片
3. **电源去耦**：添加适当的电源滤波电容
4. **保护电路**：考虑短路保护和过热保护

## 提交内容
- 完整的两级放大器电路图
- 各级电路的详细计算过程
- Python 仿真代码，包括：
  - 频率响应分析
  - 时域波形仿真
  - THD 计算和分析
- 性能总结表格（实际 vs 目标）
- 改进建议和成本分析

## 高级挑战（可选）
- 添加负反馈来进一步改善线性度和稳定性
- 设计音调控制电路（低音/高音调节）
- 实现自动增益控制（AGC）功能

## 提示
- 射极跟随器的电压增益接近 1，但电流增益很大
- 推挽输出级需要两个互补的三极管（NPN + PNP）
- THD 可以通过 FFT 分析计算
- 考虑使用运算放大器替代部分晶体管电路的可能性
