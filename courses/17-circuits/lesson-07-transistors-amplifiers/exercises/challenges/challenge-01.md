# 挑战 1：设计一个稳定的共射极放大器

## 背景
你被要求设计一个用于音频前置放大器的共射极电路。客户要求：
- 电压增益：20 倍（约 26dB）
- 频率响应：20Hz - 20kHz（±1dB）
- 输入阻抗：> 2kΩ
- 输出阻抗：< 1kΩ
- 电源电压：9V（电池供电）

## 任务要求

1. **选择合适的三极管**：从常见的小信号三极管中选择一个（如 2N3904、BC547 等），查阅其数据手册，记录关键参数。

2. **设计偏置电路**：
   - 计算 R1、R2、Rc、Re 的值
   - 确保静态工作点位于负载线中点附近
   - 考虑 β 值的变化范围（假设 β = 100 ± 50%）

3. **稳定性分析**：
   - 使用 Python 编写程序，分析当 β 从 50 变化到 150 时，静态工作点的变化
   - 计算工作点的稳定性系数 Sv = ΔIc/Ic / Δβ/β
   - 目标：Sv < 0.3

4. **频率响应优化**：
   - 添加耦合电容 C1、C2 和旁路电容 Ce
   - 计算这些电容的值以满足低频响应要求
   - 分析高频响应限制因素

5. **验证设计**：
   - 使用你编写的 Python 程序绘制直流负载线和交流负载线
   - 模拟输入 10mV 正弦波时的输出波形
   - 计算实际增益并与目标比较

## 提交内容
- 完整的电路图（手绘或使用电路设计软件）
- 所有计算过程和公式推导
- Python 仿真代码和结果截图
- 设计总结：是否满足所有要求？如有不足，如何改进？

## 提示
- 发射极电阻 Re 可以提高稳定性但会降低增益，考虑使用旁路电容 Ce
- 偏置电阻 R1、R2 的选择要平衡稳定性和功耗
- 记住 Vbe ≈ 0.7V，Vce_sat ≈ 0.2V
