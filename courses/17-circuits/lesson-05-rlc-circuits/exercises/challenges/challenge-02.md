# 挑战2：无线充电系统的谐振分析

## 背景
现代无线充电技术利用磁耦合谐振原理。发射端和接收端都包含RLC谐振电路，当两者调谐到相同频率时，能量传输效率最高。

## 任务要求
分析一个典型的无线充电系统，其中：
- 发射线圈电感 L₁ = 10μH，电阻 R₁ = 0.5Ω
- 接收线圈电感 L₂ = 8μH，电阻 R₂ = 0.4Ω  
- 耦合系数 k = 0.3（表示两个线圈的耦合程度）

### 具体要求：
1. **计算谐振频率**：选择合适的电容值C₁和C₂，使两个电路都在1MHz谐振
2. **分析耦合效应**：考虑互感M = k√(L₁L₂)，推导系统的传输函数
3. **优化设计**：如果要最大化功率传输效率，应该如何调整参数？
4. **频率分裂现象**：当耦合较强时，系统会出现两个谐振峰。计算出现频率分裂的临界耦合系数

### 提交内容：
1. 详细的计算过程和结果
2. 使用Python绘制不同耦合系数下的频率响应曲线
3. 分析频率分裂现象对实际无线充电系统的影响
4. 提出改进方案以提高系统在不同距离下的稳定性

### 提示：
- 互感 M = k√(L₁L₂)
- 耦合系统的阻抗矩阵需要考虑互感项
- 频率分裂发生在强耦合条件下（k > 1/Q）
- 实际系统中还需要考虑负载效应
