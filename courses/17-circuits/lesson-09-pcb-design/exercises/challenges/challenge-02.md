# 挑战 2：四层板电源完整性设计

## 背景
随着电子产品复杂度的增加，双层板已经无法满足高速、高密度电路的需求。四层板通过专门的电源层和地层，提供了更好的信号完整性和电源完整性。

## 任务要求

### 项目概述
设计一个基于微控制器的四层PCB，包含以下功能模块：
- STM32F103C8T6微控制器（ARM Cortex-M3）
- USB接口（用于供电和通信）
- SPI Flash存储器（8Mbit）
- I2C温度传感器
- 多个GPIO扩展接口

### 层叠结构
- **Layer 1 (Top)**: 信号层（主要放置元件和关键信号走线）
- **Layer 2 (Inner 1)**: 地平面（完整的GND平面）
- **Layer 3 (Inner 2)**: 电源平面（3.3V和5V分割区域）
- **Layer 4 (Bottom)**: 信号层（次要信号和测试点）

### 设计挑战

#### 1. 电源完整性设计
- **电源分割**：合理规划3.3V和5V电源区域，避免互相干扰
- **去耦策略**：为每个IC配置合适的去耦电容组合（100nF + 10μF）
- **过孔设计**：使用多个过孔连接不同层的电源和地网络
- **电流承载**：计算各电源网络的电流需求，确保走线/平面足够宽

#### 2. 高速信号处理
- **USB差分对**：控制100Ω差分阻抗，等长走线，避免90度拐角
- **时钟信号**：STM32内部时钟输出需要特殊处理，避免噪声耦合
- **SPI总线**：考虑信号完整性和串扰问题

#### 3. EMC优化
- **地平面完整性**：避免在高速信号下方分割地平面
- **屏蔽设计**：为敏感模拟电路提供局部屏蔽
- **I/O滤波**：在外接接口处添加适当的滤波电路

### 具体约束

#### 物理约束
- 板子尺寸：40mm × 40mm（正方形）
- 板厚：1.6mm
- 铜厚：1oz（外层）/ 0.5oz（内层）
- 最小线宽/线距：0.15mm
- 最小过孔：0.3mm孔径 / 0.6mm焊盘

#### 电气约束
- 电源纹波：< 50mV
- 信号上升时间：< 5ns
- 差分对长度匹配：±5%
- 回流路径：确保高速信号有完整的回流路径

4. **阻抗计算报告**：USB差分对和其他关键信号的阻抗分析

## 设计提示

1. **先规划电源树**：在开始布局前，先确定电源分配方案
2. **关键信号优先**：USB和时钟信号应该优先布线
3. **过孔策略**：使用盲埋孔可以节省空间，但会增加成本
4. **仿真验证**：对于关键信号，建议进行SI/PI仿真验证
