# 编程挑战 02: 电路模拟器与可视化

## 背景
为了帮助初学者更好地理解电路概念，你需要开发一个交互式的电路模拟器，能够实时显示电压、电流和电阻的关系，并提供可视化界面。

## 任务要求

### 核心功能
1. **实时计算引擎**：基于欧姆定律实现实时参数计算
2. **交互式界面**：允许用户通过滑块或输入框调整电压和电阻值
3. **动态可视化**：实时显示电流流动的动画效果
4. **数据记录**：记录用户的实验数据并生成报告

### 技术要求
- 使用 Python 和 tkinter（或其他GUI库）实现界面
- 实现欧姆定律的核心计算逻辑
- 创建直观的可视化效果来表示电流流动
- 支持数据导出功能（CSV格式）

### 功能细节

#### 界面设计
- 主窗口包含电压滑块（0-20V）和电阻滑块（1-1000Ω）
- 实时显示当前的电压、电阻、电流值
- 电路图区域显示简单的电路示意图
- 电流流动用动画箭头表示，速度与电流大小成正比

#### 计算逻辑
- 当用户调整电压或电阻时，立即重新计算电流
- 验证输入参数的有效性（电压≥0，电阻>0）
- 处理边界情况（如电阻趋近于0时的警告）

#### 可视化效果
- 电路图应包含电池、电阻和连接线
- 电流箭头的颜色可以根据电流大小变化（小电流蓝色，大电流红色）
- 添加简单的功率消耗指示器（LED亮度或颜色）

#### 数据功能
- "开始记录"按钮：开始记录实验数据
- "停止记录"按钮：停止记录并显示数据统计
- "导出数据"按钮：将记录的数据保存为CSV文件

### 示例场景
用户打开程序后：
1. 默认电压设为5V，电阻设为100Ω
2. 显示电流为0.05A，电路图中显示缓慢流动的蓝色箭头
3. 用户将电阻调到10Ω，电流变为0.5A，箭头变快变红
4. 用户点击"开始记录"，调整多个参数组合
5. 停止记录后，程序显示最大/最小/平均电流值
6. 用户可以导出完整的实验数据进行进一步分析
