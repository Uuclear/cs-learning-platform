# 编程挑战 01: 智能电路分析器

## 背景
你是一名电路工程师，需要开发一个智能电路分析工具，能够自动识别电路类型（串联或并联）并进行相应的计算。

## 任务要求

### 基本功能
1. **电路类型识别**：根据输入的电阻连接方式，自动判断是串联还是并联电路
2. **参数计算**：
   - 对于串联电路：计算总电阻、总电流、各电阻电压
   - 对于并联电路：计算等效电阻、总电流、各支路电流
3. **错误处理**：处理无效输入（如负电阻、零电压等）

### 输入格式
你的程序应该接受以下输入：
- 电源电压（伏特）
- 电路类型（"series" 或 "parallel"）
- 电阻值列表（欧姆）

### 输出格式
程序应该输出详细的分析结果，包括：
- 电路基本信息
- 计算得到的所有参数
- 功率消耗分析
- 安全性评估（如果适用）

### 示例用例

**输入示例 1（串联）**：
```
电压: 9V
类型: series
电阻: [100, 200, 300]
```

**期望输出 1**：
```
=== 串联电路分析 ===
电源电压: 9.0V
电阻列表: [100Ω, 200Ω, 300Ω]
总电阻: 600Ω
电路电流: 0.015A (15mA)
各电阻电压:
  R1 (100Ω): 1.50V
  R2 (200Ω): 3.00V  
  R3 (300Ω): 4.50V
总功率消耗: 0.135W
```

**输入示例 2（并联）**：
```
电压: 12V
类型: parallel
电阻: [6, 12, 4]
```

**期望输出 2**：
```
=== 并联电路分析 ===
电源电压: 12.0V
电阻列表: [6Ω, 12Ω, 4Ω]
等效电阻: 2.0Ω
总电流: 6.0A
各支路电流:
  支路1 (6Ω): 2.000A
  支路2 (12Ω): 1.000A
  支路3 (4Ω): 3.000A
总功率消耗: 72.0W
```
