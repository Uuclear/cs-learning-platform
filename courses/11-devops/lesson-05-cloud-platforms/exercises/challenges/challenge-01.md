# 挑战 01: 云服务成本优化分析

## 背景
你是一家初创公司的 DevOps 工程师，公司正在考虑将现有的 Web 应用迁移到云平台。应用需要以下资源：
- 2 台 Web 服务器（24/7 运行）
- 1 台数据库服务器（24/7 运行）  
- 500GB 存储空间
- 每月约 2TB 的数据传输

## 任务
1. 使用课程中的成本估算示例，为这个工作负载在 AWS、GCP 和 Azure 上进行成本估算
2. 考虑不同的优化策略：
   - 预留实例 vs 按需付费
   - 不同的存储类别（标准 vs 低频访问）
   - 不同的地理区域
3. 创建一个详细的成本对比报告，包括：
   - 各平台的基础成本
   - 优化后的成本
   - 节省的百分比
   - 推荐的平台和理由

## 要求
- 使用 Python 编写自动化脚本进行成本计算
- 考虑至少 3 种不同的优化组合
- 报告应包含图表或表格形式的对比结果
- 提供具体的实施建议

## 扩展挑战
- 考虑一年和三年的成本差异
- 分析如果流量增长 50% 对成本的影响
- 评估混合云方案的成本效益
