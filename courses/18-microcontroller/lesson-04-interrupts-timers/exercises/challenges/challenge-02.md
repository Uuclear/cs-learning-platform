# 挑战 02: 精确电机控制系统

## 背景
你正在为一个机器人项目开发精确的电机控制系统。系统需要同时控制4个直流电机，每个电机都需要独立的PWM信号和位置反馈。

## 要求
使用定时器和中断实现一个高精度的电机控制系统。

### 功能需求
1. **PWM生成**：为每个电机生成独立的PWM信号（频率10kHz，占空比0-100%可调）
2. **编码器读取**：通过外部中断读取电机编码器脉冲，计算实时转速
3. **PID控制**：在定时器中断中实现PID算法，调整PWM占空比
4. **故障保护**：过流、过热等紧急情况下的快速响应

### 技术约束
- PWM频率必须稳定在10kHz ± 1%
- 编码器中断响应时间 < 5μs
- PID控制周期：1ms
- 紧急停止响应时间 < 100μs
- 系统时钟：168MHz（STM32F4系列）

### 实现步骤
1. **定时器配置**：
   - 配置高级定时器生成4路PWM输出
   - 计算预分频值和自动重载值以获得10kHz PWM
   - 设置更新中断用于PID控制

2. **外部中断配置**：
   - 配置4个GPIO引脚用于编码器输入
   - 设置上升沿触发的外部中断
   - 实现高效的脉冲计数算法

3. **中断优先级设计**：
   - 紧急停止：最高优先级（0）
   - 编码器中断：高优先级（1）
   - PWM更新中断：中优先级（2）
   - 其他系统中断：低优先级（3+）

4. **PID算法实现**：
   - 在1ms定时器中断中执行PID计算
   - 使用定点数运算避免浮点开销
   - 实现积分限幅防止积分饱和

5. **系统集成**：
   - 主循环处理用户命令和状态显示
   - ISR只负责数据采集和PWM更新
   - 实现平滑的启动和停止逻辑

### 验证标准
- PWM频率精度：10kHz ± 1%
- 编码器计数准确无丢失
- PID控制稳定，无振荡
- 紧急停止功能可靠
- 系统在满负载下稳定运行

### 扩展挑战（可选）
1. 实现电机同步控制（多电机协调运动）
2. 添加加速度限制功能
3. 实现位置模式（目标位置控制）
4. 优化内存使用，支持更多电机通道

### 提交内容
- 完整的硬件配置代码（时钟、GPIO、定时器、中断）
- 中断服务程序实现
- PID控制算法代码
- 系统测试报告（波形图、性能数据）
- 设计决策说明文档

> **提示**：考虑使用DMA来减轻CPU负担，对于高频PWM可以考虑使用定时器的互补输出功能。
