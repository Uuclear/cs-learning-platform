# 挑战 2：LED 电阻计算器 Web 应用

## 背景
在课程中，我们创建了一个命令行版本的 LED 限流电阻计算器。现在让我们将其升级为一个交互式的 Web 应用，让更多人能够方便地使用！

## 任务要求
使用 Python 创建一个简单的 Web 应用，让用户可以通过浏览器计算 LED 限流电阻。

### 基本功能
1. 创建一个 **Web 界面**，包含以下输入字段：
   - 电源电压（V）
   - LED 颜色（下拉选择：红、橙、黄、绿、蓝、白、紫）
   - 或直接输入 LED 电压降（V）
   - 期望工作电流（mA）

2. 实时计算并显示：
   - 所需的精确电阻值
   - 推荐的标准电阻值（E12 系列）
   - 实际工作电流
   - 安全性评估（绿色/黄色/红色指示）

3. 提供 **电路图预览**（可以是静态图片或简单 SVG）

### 技术要求
- 使用 **Flask** 或 **Streamlit** 框架
- 代码结构清晰，模块化设计
- 包含适当的错误处理和用户反馈
- 响应式设计，适配不同屏幕尺寸

### 高级功能（选做）
4. 添加 **历史记录** 功能，保存用户的计算历史
5. 实现 **批量计算**，支持同时计算多个 LED 的参数
6. 添加 **导出功能**，可以将结果导出为 PDF 或 CSV
7. 集成 **Arduino 代码生成器**，根据计算结果自动生成相应的 Arduino 代码

## 文件结构要求
```
led-calculator/
├── app.py              # 主应用文件
├── static/
│   ├── css/
│   │   └── style.css   # 样式文件
│   └── images/
│       └── circuit.png # 电路图
├── templates/
│   └── index.html      # 主页面模板
└── utils/
    └── calculator.py   # 计算逻辑模块
```

## 提示
- 将课程中的 `example-01.py` 中的计算逻辑提取到独立的模块
- 使用 Bootstrap 或类似的 CSS 框架快速构建美观的界面
- 考虑使用 JavaScript 实现实时计算，减少服务器请求
- 对于电路图，可以使用简单的 SVG 或准备几张静态图片

## 评估标准
- ✅ Web 应用能够正常运行并提供准确的计算结果
- ✅ 用户界面友好，输入验证完善
- ✅ 代码结构合理，遵循良好的编程实践
- ✅ （高级功能）历史记录功能稳定可靠
- ✅ （高级功能）Arduino 代码生成准确可用
