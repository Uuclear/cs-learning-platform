# 挑战 01：智能风扇控制系统

## 背景
你正在设计一个智能电脑机箱风扇控制系统。系统需要根据CPU温度自动调节风扇转速，同时避免频繁的转速变化造成噪音和磨损。

## 要求
1. **温度读取**：模拟从温度传感器读取CPU温度（范围30-90°C）
2. **PWM映射**：将温度线性映射到PWM占空比（0-255范围）
   - 30°C → 0 PWM (风扇停止)
   - 60°C → 128 PWM (50%速度)
   - 90°C → 255 PWM (全速)
3. **平滑控制**：实现转速平滑过渡，避免突然变化
4. **死区处理**：当温度变化小于2°C时，不改变风扇转速
5. **安全保护**：温度超过85°C时，强制全速运行

## 扩展挑战
- 添加LED指示灯显示当前温度级别（绿色<50°C，黄色50-70°C，红色>70°C）
- 实现风扇启动时的软启动（逐渐增加到目标速度）
- 添加按键手动覆盖自动模式

## 提交要求

- 包含电路连接图（可以手绘或使用Fritzing）
- 编写测试计划验证所有功能
