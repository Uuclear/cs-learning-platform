# 挑战 1：实现量子傅里叶变换

## 背景
量子傅里叶变换（QFT）是许多重要量子算法的核心组件，包括Shor算法。它能够将量子态从时域转换到频域。

## 任务
使用Python实现量子傅里叶变换的概念性模拟：

1. 创建一个函数 `quantum_fourier_transform(amplitudes)`，接受复数振幅列表作为输入
2. 实现QFT的数学公式：对于N维向量，第k个输出为：
   ```
   QFT(x)_k = (1/√N) * Σ_{j=0}^{N-1} x_j * e^(2πijk/N)
   ```
3. 确保输入向量长度为2的幂次（如2, 4, 8, 16...）
4. 返回变换后的复数振幅列表

## 提示
- 使用 `cmath.exp()` 计算复指数
- 注意归一化因子 `1/√N`
- 可以先在小规模（N=4）上测试

## 验证
- 输入均匀叠加态，验证输出是否正确
- 输入单个基态 |0>，验证QFT结果
