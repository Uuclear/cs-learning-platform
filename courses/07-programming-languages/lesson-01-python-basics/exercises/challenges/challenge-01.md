## 挑战1: 猜数字游戏 ⭐⭐

### 题目描述
创建一个猜数字游戏，程序随机生成一个1到100之间的整数，玩家需要猜测这个数字。程序会根据玩家的猜测给出提示（"太大了"、"太小了"或"猜对了"），直到玩家猜中为止。

### 具体要求
- 使用 `random.randint(1, 100)` 生成随机数
- 使用 `input()` 函数获取玩家输入
- 处理无效输入（非数字输入）
- 记录玩家猜测的次数
- 游戏结束后显示总猜测次数
- 询问玩家是否要再玩一次

### 示例输出
```
我想到了一个1到100之间的数字，你能猜出来吗？
请输入你的猜测: 50
太大了！
请输入你的猜测: 25
太小了！
请输入你的猜测: 37
猜对了！你用了3次猜测。
还要再玩一次吗？(y/n): n
谢谢游戏！
```

### 提示
- 使用 `try-except` 处理 `ValueError` 异常
- 使用 `while` 循环控制游戏流程
- 使用布尔变量控制游戏是否继续
