# 编程挑战 1：工厂模式实现 🌟

## 背景故事
你正在开发一个动物园管理系统，需要能够动态创建不同类型的动物。每种动物都有自己的叫声，但你不希望在客户端代码中直接使用具体的动物类。

## 任务要求
1. 创建一个 `Animal` 抽象基类，包含 `speak()` 抽象方法
2. 实现至少三种具体的动物类（如 `Dog`, `Cat`, `Duck`），每个都有不同的叫声
3. 创建一个 `AnimalFactory` 工厂类，能够根据字符串参数创建对应的动物实例
4. 编写测试代码，验证工厂能够正确创建不同类型的动物

## 示例输出
```
Dog: 汪汪！
Cat: 喵喵！
Duck: 嘎嘎！
```

## 提示
- 使用 Python 的 `abc` 模块来定义抽象基类
- 工厂方法应该是静态方法或类方法
- 考虑如何处理无效的动物类型输入

## 难度：⭐（简单）
