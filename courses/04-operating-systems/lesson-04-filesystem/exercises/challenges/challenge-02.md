# 编程挑战 2: 智能文件清理器 ⭐⭐⭐

## 题目描述

编写一个智能文件清理程序，能够识别并安全删除重复文件、临时文件和大文件。

## 具体要求

### 功能1: 重复文件检测
- 扫描指定目录，找出内容完全相同的文件
- 对于重复文件，保留一个，其他标记为可删除
- 使用文件大小预筛选，然后计算文件哈希值进行精确比较

### 功能2: 临时文件清理
- 识别常见的临时文件（如 `.tmp`, `.temp`, `~`, `.bak` 等）
- 根据最后修改时间，删除超过7天的临时文件

### 功能3: 大文件管理
- 识别大于指定大小（默认100MB）的文件
- 显示这些文件的信息，询问用户是否要删除

### 用户交互
- 程序应该先显示分析结果，不自动删除任何文件
- 提供交互式确认删除操作
- 支持命令行参数配置：
  - `-d, --directory`: 指定扫描目录（必需）
  - `-s, --size`: 大文件阈值（默认100MB）
  - `-a, --auto`: 自动模式（不询问，直接删除）
  - `-n, --dry-run`: 模拟运行，只显示会删除什么

## 示例用法

```bash
# 分析当前目录
python file_cleaner.py -d .

# 分析并自动删除临时文件（大于50MB的大文件）
python file_cleaner.py -d /home/user/downloads -s 50M -a

# 模拟运行，查看会删除什么
python file_cleaner.py -d . -n
```

## 技术要求

1. 使用 `hashlib` 计算文件哈希值（MD5或SHA256）
2. 实现高效的重复文件检测算法（避免不必要的哈希计算）
3. 正确处理文件路径和权限
4. 提供详细的日志输出
5. 代码结构清晰，模块化设计

## 安全考虑

- **绝不**在没有用户确认的情况下删除重要文件
- 对于重复文件，保留最早创建或最近修改的文件
- 记录所有删除操作到日志文件
- 处理中断信号（Ctrl+C），确保程序安全退出

## 扩展挑战（可选）

- 添加图形用户界面
- 支持网络驱动器扫描
