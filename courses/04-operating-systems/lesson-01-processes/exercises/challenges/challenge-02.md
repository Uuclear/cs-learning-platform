# 编程挑战2: 进程家族树分析器

## 背景
在复杂的系统中，理解进程间的父子关系对于调试和性能分析非常重要。

## 任务要求
实现一个进程家族树分析工具，能够：
1. 接受一个进程PID作为输入
2. 构建完整的进程家族树（包括所有祖先和后代）
3. 分析并报告家族树的统计信息
4. 可视化显示家族树结构

## 具体要求
- 实现以下功能函数：
  - `get_ancestors(pid)`: 获取指定进程的所有祖先进程
  - `get_descendants(pid)`: 获取指定进程的所有后代进程
  - `analyze_family_tree(pid)`: 分析家族树并返回统计信息
  - `print_family_tree_visual(pid)`: 以可视化方式打印家族树
- 统计信息应包括：
  - 家族总进程数
  - 树的最大深度
  - 平均每个进程的子进程数
  - 家族中资源消耗最高的进程
- 处理循环引用和权限问题
- 提供友好的命令行界面
- 代码要有完整的错误处理和中文注释

## 示例使用
```bash
python family_analyzer.py --pid 12345 --stats --visual
```
