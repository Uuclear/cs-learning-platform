# 编程挑战1：实现SJF调度算法 ⭐⭐

## 背景
最短作业优先（Shortest Job First, SJF）调度算法选择执行时间最短的进程先执行，可以最小化平均等待时间。

## 任务
实现一个非抢占式的SJF调度算法，要求：

1. 创建`Process`类，包含进程ID、到达时间、执行时间、完成时间、周转时间和等待时间属性
2. 实现`sjf_scheduling(processes)`函数，返回调度后的进程列表、平均周转时间和平均等待时间
3. 处理CPU空闲的情况（当没有进程就绪时）
4. 按照示例格式输出调度过程和性能统计

## 输入示例
```python
processes = [
    Process(1, 0, 8),
    Process(2, 1, 4),
    Process(3, 2, 9),
    Process(4, 3, 5),
]
```

## 输出要求
- 调度过程表格，显示每个进程的详细信息
- 平均周转时间和平均等待时间（保留2位小数）

## 提示
- 在每个时间点，找出所有已到达且未完成的进程中执行时间最短的那个
- 注意处理多个进程同时到达的情况
- 参考FCFS和优先级调度的实现方式
